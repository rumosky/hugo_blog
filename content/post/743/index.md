## 说明

前两天购买了阿斯加特 ddr5 6400M 32G*2套条，但是问题很多，然后换了光威天策 ddr5 6400M 32G*2套条，记录一下。

### 介绍

![光威天策64G套条](https://cdn.rumosky.com/usr/uploads/2024/04/2761395916.png)

这个内存问题更多，比起阿斯加特，第一次开机主板黄灯故障，重启了三次，才进入系统，结果还不到一分钟，就蓝屏了，然后开机之后，又陆续蓝屏了三次，基本上处于不能用的状态。

趁电脑不注意，打开TM5，进行了唯一的一波测试，结果如下：

![光威天策64G测试](https://cdn.rumosky.com/usr/uploads/2024/04/3454659482.png)

直接爆红，错误数65，比起阿斯加特是差远了

**即使降低频率到默认的5600，也还是报错，只是少了一半**

```bash
# 电脑配置
CPU：i5-13490F
主板：华硕B760 TUF GAMING WIFI D5
显卡：蓝戟Intel Arc A770 Photon 16G OC 2400MHz超频版
内存：光威天策D5 6400 32G*2
电源：长城X7 750W
硬盘：致态7100 2TB
```

## 结论

这个内存条跟我的硬件兼容性极差，基本上处于不能用的状态，但这个比阿斯加特便宜一百五，1351买的，还可以，结果在我发文章的时候，内存已经涨价了，硬盘也涨价了。可以说，64G内存，稳定的话，还是要D4，或者D5配合Z系列主板。
