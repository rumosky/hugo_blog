## 说明

最近因为玩天际线2，想升级到64G内存，购买了阿斯加特 ddr5 6400M 32G*2套条，但是问题很多，记录一下。

### 介绍

![阿斯加特64G套条](https://cdn.rumosky.com/usr/uploads/2024/04/4247091893.png)

上图就是我买的D5 6400频率的内存，这个内存上机第一次开机自动重启，主板报黄灯，重启之后正常进入系统，打开LOL，打了一局，刚结束，直接蓝屏，如下图：

![蓝屏](https://cdn.rumosky.com/usr/uploads/2024/04/1694011239.jpg)

然后我赶紧掏出我的TM5，进行了好几拨测试，最差的一次如下图：

![阿斯加特64G测试](https://cdn.rumosky.com/usr/uploads/2024/04/4084623089.png)

还好，就一块红了，然后用了快一周，再也没有蓝屏过，之后又测试了两次，一次啥问题都没有，还有一次也跟上图结果一样。

**但是降低频率到默认的5600，一点问题都没有，只是有点亏，毕竟买的是6400的**

```bash
# 电脑配置
CPU：i5-13490F
主板：华硕B760 TUF GAMING WIFI D5
显卡：蓝戟Intel Arc A770 Photon 16G OC 2400MHz超频版
内存：阿斯加特D5 6400 32G*2
电源：长城X7 750W
硬盘：致态7100 2TB
```

## 结论

这个内存条跟我的硬件兼容性还可以，基本上使用不会出什么大问题，但D5内存，还是不建议用64G的，48G稳定一些。而且，其实大部分人都用不到64G，32G都绰绰有余。非必要还是不要上64G了。而且这个内存也不便宜，1499买的。兼容性问题可能也是主板问题，换成Z690可能好一些，而且，本人手动超频到7200也基本上可以用，没啥大问题，就是有点心虚。
