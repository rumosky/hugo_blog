## 说明

最近需要安装B站的电视版客户端，但是即便在系统上开启了允许安装第三方应用，还是提示监管要求，不能安装。试了别的手机连接电视安装的方式，都不好，后来找到一个方式很容易安装，记录一下。

## 步骤

1，首先，在设置里面，点击`关于 - 产品型号`六次，就可以打开开发者模式；
2，在账号与安全页面，打开`ADB调试`
3，在电脑上下载ADB工具，地址是：https://adbdownload.com/
4，在电脑上解压这个软件，并把这个软件目录设置为环境变量，同时把需要安装的apk文件放在这个目录下
5，打开终端cmd，运行`adb connect 192.168.xxx.xxx`，IP请自行替换为自己的电视IP，同时在电视上点击允许，确定。
6，执行`adb devices`显示下面的内容即为连接正常

```bash
List of devices attached
192.168.xxx.xxx:5555      device
```

若显示下面的内容，则表示你没有在电视上点击允许。

```bash
List of devices attached
192.168.xxx.xxx:5555      unauthorized
```

7，执行这个命令` adb install .\xxx.apk`安装apk，显示下面的内容则表示安装成功

```bash
Performing Streamed Install
Success
```

### 注意

我的电视系统版本是澎湃系统3.0.24，安卓版本好像是14，安装成功之后，打开这个软件，还是会提示非官方应用，需要再点一下运行才可以进入软件，比电视上自己下载的要多一步操作。
